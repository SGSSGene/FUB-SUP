# Differentialgeometrie I
|                                    |   |
|------------------------------------|---|
|**Modul**                           | Differentialgeometrie I |
|**Hochschule/Fachbereich/Institut** | Freie Universität Berlin/Mathematik und Informatik |
|**Modulverantwortung**              | Dozentinnen und Dozenten des Moduls |
